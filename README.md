# Laravel CRUD with Authentication & Role System

A professional Laravel CRUD (Create, Read, Update, Delete) application with Authentication and Role-Based Access Control (Admin & User).

---

## 🚀 Features

- User Registration & Login
- Secure Authentication System
- Role-Based Access (Admin & User)
- Protected Routes using Middleware
- Full CRUD Operations
- Form Validation
- Clean Blade UI
- MVC Architecture

---

## 🛠️ Tech Stack

- PHP 8+
- Laravel 10+
- MySQL
- Blade Template Engine
- Bootstrap / Tailwind CSS
- Vite

---

## 👥 User Roles

### 🔹 Admin
- Full CRUD access
- Manage all records
- Access protected routes

### 🔹 User
- Limited access
- View-only permissions
